# SRMS - School Resource Management System

A comprehensive school resource management system by WeGEM.

## Features
- Book management and lending
- Furniture allocation
- Teacher management
- Class list management
- QR code generation
- Staff chat system
- Audit logging
- Multiple wallpaper themes

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Supabase
- Deployment: Vercel

## Setup
1. Clone the repository
2. Deploy to Vercel
3. Configure Supabase database
